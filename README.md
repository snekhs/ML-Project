DETECTION AND CLASSIFICATION OF ACUTE LYMPHOBLASTIC LEUKEMIA USING MACHINE LEARNING ALGORITHMS

Acute Lymphoblastic Leukemia is a type of Blood Cancer affecting White Blood Cell and Bone Marrow, Primarily seen in Children. Machine Learning offers solution by automating this process and improving diagnostic accuracy.

OBJECTIVE - To develop a ML Model capable of detecting and classifying Acute Lymphoblastic Leukemia from Microscopic images of Blood Samples.
METHODOLOGY:
1. DATA COLLECTION: Dataset of Microscopic Blood sample images - LINK
2. PREPROCESSING:
3. Resizing images, normalizing pixel values
4. MODEL SELECTION AND TRAINING:
5. SVM, Random Forest, KNN, Naive Bayes, XGBoost
6. EVALUATION METRICS:
7. Precision, recall, f1-score, Accuracy
8. OVERALL COMPARISON OF EVALUATION METRICS: PIC
